## Install

```sh
pip install git+https://github.com/hananana/iris.git
```

## How to setup project

```sh
PYTHON_CONFIGURE_OPTS="--enable-framework" pyenv install 3.6.0
pyenv virtualenv 3.6.0 iris
pyenv local iris
pip install -r requirements.txt
```
